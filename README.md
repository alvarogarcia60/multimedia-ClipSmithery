# Mini-Netflix AI (Multimedia UCLM)

---

## Descripción del Proyecto

Este proyecto es una aplicación multimedia diseñada para simular una plataforma de streaming a pequeña escala (Mini-Netflix). 

El sistema incluye una serie de funcionalidades de procesamiento de medios asistidas por Inteligencia Artificial (AI), cubriendo el ciclo completo de ingesta y presentación de contenido:

* **Subida y Compresión de Vídeo:** Permite la carga de nuevos vídeos y su optimización mediante algoritmos de compresión.
* **Extracción y Transcripción de Audio:** Procesa el vídeo para aislar la pista de audio y la convierte en texto utilizando modelos de transcripción.
* **Generación Automática de Resúmenes:** Aplica técnicas de AI para generar resúmenes concisos del contenido transcrito.
* **Reproductor Multimedia Personalizado:** Implementación de un reproductor propio para gestionar la visualización del contenido.
* **Actualizaciones Dinámicas:** La información de la plataforma (catálogo, metadatos) se gestiona y actualiza dinámicamente a través de archivos **JSON**.
